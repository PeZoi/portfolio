# SKILL: UI/UX Pro Max (for Antigravity)

## ROLE
Antigravity must ensure that every component and page created possesses exceptionally high aesthetics, delivering a "Premium" experience for the USER. Absolutely do not create minimalistic (MVP) or bare-bones interfaces.

## DESIGN & AESTHETIC STANDARDS
1. **Dynamic Design**:
   - Integrate hover states and micro-animations (e.g., using CSS transitions or Framer Motion if available in the project).
   - Ensure interactions feel alive and highly responsive.
2. **Visual Excellence**:
   - Avoid boring basic colors (pure red, blue, green). Utilize smooth HSL palettes, sleek dark modes, and glassmorphism.
   - Use modern typography (Inter, Roboto, Outfit).
3. **UI Mockups**:
   - When discussing interfaces with the USER, utilize the `generate_image` tool to create design mockups instead of just describing them.
4. **StitchMCP Integration**:
   - If the project uses the StitchMCP server, use its tools (`apply_design_system`, etc.) to ensure accurate designs that align with the Design System.

## STRICT RULE
If the interface looks simple, bare, or ugly, it is considered a FAILURE. Always push design limits to the maximum level according to your system requirements (Antigravity).
